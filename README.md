# -Strong-Password-and-Evaluate-Its-Strength-Task-6
This project involves creating and testing passwords with different complexity levels using online strength checker tools. It helps understand how length, symbols, and character variety improve security, while highlighting common attacks and best practices for strong authentication.

## Objective
To create passwords with different complexity levels and test their strength using a password evaluation tool.

## Tools Used
- Online Password Strength Checker
- Web Browser
- GitHub

## Description
This task focuses on creating passwords with different complexity levels and evaluating their strength using an online password checker. The goal is to understand how password length, uppercase letters, lowercase letters, numbers, and special characters improve security. It also explains why weak passwords are vulnerable to brute force and dictionary attacks.

In this task, different passwords were created using a mix of uppercase letters, lowercase letters, numbers, and special characters. Each password was tested in an online password strength checker to analyze its score, complexity, and resistance against attacks. The test results showed that short and simple passwords are weak and easier to crack, while longer passwords with mixed characters provide stronger protection.

The project also covered best practices such as avoiding common words, not using personal information, using passphrases, and enabling multi-factor authentication for extra protection.

## Steps Performed
1. Created multiple passwords with different complexity levels.
2. Used uppercase, lowercase, numbers, and symbols.
3. Tested each password in an online strength checker.
4. Recorded the results and feedback.
5. Compared weak, medium, and strong passwords.
6. Studied common password attack methods.
7. Summarized strong password best practices.

## Result Analysis

| Sample Password       | Complexity Level | Strength Result | Remarks |
|-----------------------|------------------|-----------------|---------|
| password123           | Low              | Weak            | Common and predictable pattern |
| Dhanush@123           | Medium           | Moderate        | Better than simple passwords, but still predictable |
| T!ger#Sky_2026        | High             | Strong          | Good mix of uppercase, lowercase, numbers, and symbols |
| River!Moon#85@Star    | Very High        | Very Strong     | Long, complex, and difficult to guess |

## Results
- Weak passwords scored low because they were short and easy to predict.
- Medium passwords improved when uppercase letters and numbers were added.
- Strong passwords had better ratings because of increased length and symbol usage.
- Very strong passwords were long, unique, and used multiple character combinations.
- Password length and randomness greatly increased overall security.
